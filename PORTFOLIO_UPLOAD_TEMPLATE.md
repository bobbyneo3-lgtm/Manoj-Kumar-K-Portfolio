## 🛠 How to get the "Direct Link" from Google Drive

For your images and videos to show up on the website, follow these exact steps:

1. **Upload** your file to Google Drive.
2. **Right-click** the file > **Share** > **Share**.
3. Change "General access" to **"Anyone with the link"**.
4. **Copy the link**. It will look like this:
   `https://drive.google.com/file/d/1A2B3C4D5E6F7G8H9I0J/view?usp=sharing`
5. **Extract the ID**: The ID is the long code between `/d/` and `/view`.
   In the example above, the ID is: `1A2B3C4D5E6F7G8H9I0J`
6. **Create the Direct Link**: Use the ID in this format:
   `https://lh3.googleusercontent.com/d/1A2B3C4D5E6F7G8H9I0J`

## 📏 Recommended Specs
- **Images:** `.jpg` or `.png`, size around `800x1000px` for thumbnails.
- **Videos:** `.mp4`, size under `15MB`, resolution `1080p`.
- **Names:** Use clear titles like "High-End Product Ad" instead of "Video_Final_v2".
